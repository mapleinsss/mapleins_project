## 流
    1. Collection 提供了新的 stream() 方法
    2.流不存储值，通过管道的方式获取值
    3.本质是函数式的，对流的操作会生成一个结果，不过并不会修改底层的数据源，集合可以作为流的底层数据源
    4.延迟查找，很多流操作（过滤、映射、排序等）都可以延迟实现
